Dijkstra's Path finding:

My project is a graphical implementation of Dijkstra's pathfinding algorithm using Python's Pygame library. The game features a grid where users can set start and target points, as well as obstacles. The algorithm visualizes the pathfinding process in real-time, highlighting visited nodes, queued nodes, and the final path from start to target. The project includes interactive elements for users to draw walls with the mouse and set the target point. If no path is found, a message box alerts the user.

Weather Checking web Appplication:

My project is a React-based weather app built using Vite as part of the MERN stack. The app fetches weather data from the OpenWeatherMap API, displaying the current temperature, location, weather description, feels like temperature, humidity, and wind speed. Users can enter a location in the input field, and upon pressing Enter, the app retrieves and displays the relevant weather data. The temperature is converted from Kelvin to Celsius for better user understanding. The design includes top and bottom sections for displaying weather details comprehensively.

Smart Green House(IOT project):

This project demonstrates a comprehensive temperature monitoring system using the RuggedBoard A5D2X and STM32 F446RE microcontrollers. The system effectively monitors temperature, compares it to a critical threshold, and activates a relay to control an external device in response to high temperature conditions. The use of master-slave communication enables efficient data exchange and control between the different components.
